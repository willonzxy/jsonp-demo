# jsonp 百度首页（静态页示例）

一个**纯静态**的前端练习项目：仿“百度首页”基础布局，并通过 **JSONP** 调用百度联想词接口实现搜索建议。

### 功能

- **搜索建议（JSONP）**：输入关键词后，动态插入 `<script>` 请求百度联想词接口并渲染下拉列表
- **键盘交互**：↑/↓ 选择建议项、Enter 直接跳转搜索
- **鼠标交互**：点击建议项跳转搜索；“更多产品”悬浮显示右侧栏
- **页面布局**：顶部导航、搜索区、内容区、右侧“实时热点”等静态布局与样式

### 运行方式

这是静态页面项目，无需安装依赖。

- **直接打开**：用浏览器打开 `jsonp_baidu.html`
- **推荐（本地起服务）**：

```bash
python3 -m http.server 8080
```

然后访问：`http://localhost:8080/jsonp_baidu.html`

### 实现要点（简述）

- **JSONP 联想词接口**：在 `js/jsonp.js` 中，根据输入值动态创建 `<script>`：
  - 请求：`https://sp0.baidu.com/.../su?wd=<keyword>&cb=myJsonp`
  - 回调：全局函数 `myJsonp(...)` 读取返回的 `s` 数组并渲染到 `#list`
- **搜索跳转**：回车/点击建议项会跳转到 `https://www.baidu.com/s?wd=<keyword>`

### 注意事项

- **需要联网**：联想词接口与部分图标资源来自外网；离线将无法返回建议数据。
- **接口可能变更**：第三方接口（百度）若变更参数/域名，联想词功能可能失效，需要更新 `js/jsonp.js` 的请求地址。
